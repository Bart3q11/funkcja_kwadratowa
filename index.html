Funkcja Kwadratowa

⨍(x) = ax2 + bx + c

Czym jest funkcja kwadratowa?

Funkcja kwadratowa to funkcja którą można zapisać wzorem:

⨍(x) = ax2 + bx + c

gdzie:

a – współczynnik przy x2(określa, czy ramiona paraboli są skierowane w górę a > 0 czy w dół a < 0 ) a ≠ 0,

b – współczynnik liniowy,

c – wyraz wolny (przesunięcie funkcji na osi Y).

Wykresem funkcji jest parabola o wierzchołku W = (p; q), gdzie

  • p = - b 2 a
  • q = - Δ 4 a

Postacie funkcji kwadratowej

Funkcje kwadratowe możemy zapisać w różnych postaciach takich jak:

Postać ogólna funkcji kwadratowej:

⨍(x) = ax2 + bx + c
gdzie a ≠ 0; b, c ∈ ℝ, x ∈ ℝ

Postać kanoniczna funkcji kwadratowej:

⨍(x) = a(x - x1)(x - x2)
gdzie: x1, x2 to miejsca zerowe.

Postać iloczynowa funkcji kwadratowej:

⨍(x) = a(x - p)2 + q
gdzie: p oraz q to współrzędne wierzchołka paraboli W.

Miejsca zerowe

Z postaci ogólnej funkcji kwadratowej można obliczyć wyróżnik trójmianu kwadratowego, czyli Deltę:

Δ = b2 - 4ac

Δ > 0, to funkcja kwadratowa ma dwa miejsca zerowe;

x 1 = - b - Δ 2 a

x 2 = - b + Δ 2 a

Δ = 0, to funkcja kwadratowa ma dokładnie jedno miejsce zerowe.

x 1 = x 2 = - b 2 a

Δ < 0, to funkcją kwadratowa nie ma miejsc zerowych.

Δ ≧ 0, to funkcją kwadratową możemy zapisać w postaci iloczynowej

Wzory Viete'a

Jeśli Δ ≧ 0, to

  • x 1 + x 2 = - b a
  • x 1 ⋅ x 2 = c a
